# mergekonflict.com

## Project Overview

**mergekonflict.com** is a digital hub for the drum and bass DJ "merge konflict". The project is built as a high-performance website designed to showcase his music (specifically dark techstep and amen-heavy dnb), upcoming shows, and social presence. It aims to provide a "dark, moody" aesthetic with aggressive accent colors, reflecting the artist's musical style.

The project is currently in its early initialization phase, using **Next.js** (App Router) and **React 19**.

## Building and Running

The project uses `pnpm` as the primary package manager.

### Key Commands

- **Development Server:** `pnpm dev`
- **Build for Production:** `pnpm build`
- **Start Production Server:** `pnpm start`
- **Linting:** `pnpm lint`
